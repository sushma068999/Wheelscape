# Wheelscape
A decision-making web app that helps users create custom wheels and spin them to make random choices.

Live demo: [link]

## Features
- Create custom decision wheels
- Interactive wheel creation popup
- Modern glassmorphism UI
- Clean and aesthetic interface
- Responsive layout
- Decorative hero section
